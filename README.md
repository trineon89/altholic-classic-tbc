# Altoholic Classic TBC

This repository contains the Altoholic addon suite and DataStore modules for
World of Warcraft Classic: The Burning Crusade.

## Contents

- `Altoholic` and related modules: user interface, summary views, and search tools.
- `DataStore` modules: account-wide storage for characters, inventory, quests,
  reputations, talents, and more.
- `AddonFactory`: shared addon utilities.

## Usage

Install the addon folders into your WoW Classic AddOns directory, then enable
Altoholic and the DataStore modules you want to use.
